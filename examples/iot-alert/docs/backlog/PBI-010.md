## ストーリー

システム管理者として、ユーザー認証でアクセスを制御したい。なぜなら許可されたユーザーのみがシステムを利用できるようにする必要があるから。

## 受入条件

- [ ] ログイン画面（SCR-001）でメールとパスワードを入力してログインできる
- [ ] 未認証ユーザーは全画面へのアクセスが拒否され、ログイン画面にリダイレクトされる
- [ ] パスワードは8文字以上、英数記号混合が必須
- [ ] ログイン失敗時にエラーメッセージが表示される（「メールアドレスまたはパスワードが正しくありません」）
- [ ] ログアウトできる
- [ ] セッションタイムアウト後に再ログインが求められる
