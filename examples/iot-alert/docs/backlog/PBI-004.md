## ストーリー

保全担当者として、警告以上のアラートをメールでも受け取りたい。なぜならSlackを見逃しても気づけるようにしたいから。

## 受入条件

- [ ] 閾値判定で「警告」以上になった場合、メールが送信される（BR-005）
- [ ] 「注意」レベルではメールは送信されない（Slackのみ）
- [ ] メール本文に設備名、センサー名、現在値、閾値、通知レベル、発生時刻が含まれる
- [ ] メール本文に設備詳細画面へのリンクが含まれる
- [ ] 「異常」レベルの場合、全員にメール送信される（BR-006）
- [ ] 担当グループのメンバーにのみメール送信される（担当外には送信されない）
