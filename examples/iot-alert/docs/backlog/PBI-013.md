## ストーリー

システム管理者として、誰がいつ何を見たか・操作したかのログを自動記録したい。なぜなら規制対応でアクセスログの保持が求められているから。

## 受入条件

- [ ] ユーザーのログイン/ログアウトがログに記録される
- [ ] 画面閲覧（ダッシュボード、設備詳細）がログに記録される
- [ ] 設定変更（閾値変更、通知グループ変更、ユーザー管理操作）がログに記録される
- [ ] 各ログにユーザーID、操作日時、操作内容、対象リソースが含まれる
- [ ] ログは改ざんされない形式で保存される（CloudWatch Logs等）
