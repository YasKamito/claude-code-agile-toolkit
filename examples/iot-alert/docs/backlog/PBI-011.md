## ストーリー

システム管理者として、ユーザーの追加・削除・権限設定をしたい。なぜならアクセス管理が必要だから。

## 受入条件

- [ ] ユーザー一覧が表示される（名前、メール、権限、最終ログイン）
- [ ] 新規ユーザーを追加できる（名前、メール、初期パスワード、権限）
- [ ] ユーザーの権限を変更できる（管理者/一般ユーザー）
- [ ] ユーザーを削除（無効化）できる
- [ ] 管理者権限を持つユーザーのみがこの画面にアクセスできる
