## ストーリー

システム管理者として、クラウド環境へのアクセスをVPN経由のみに制限したい。なぜならセキュリティポリシーで直接インターネット接続は許可できないから。

## 受入条件

- [ ] クラウド環境へのアクセスがVPN経由でのみ可能になっている
- [ ] VPN外からのHTTPSアクセスが拒否される
- [ ] VPN接続後、ダッシュボード等の画面に正常にアクセスできる
- [ ] PLCゲートウェイからクラウドへのデータ送信もVPN経由で行われる
- [ ] VPN接続手順書が作成されている
