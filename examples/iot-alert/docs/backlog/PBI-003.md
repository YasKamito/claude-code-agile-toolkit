## ストーリー

保全担当者として、閾値超えのアラートをSlackで受け取りたい。なぜなら巡回していなくても異常に気づけるようになりたいから。

## 受入条件

- [ ] 閾値判定でレベル遷移が発生し「注意」以上になった場合、Slackに通知が送信される
- [ ] 通知メッセージに以下が含まれる: 設備名、センサー名、現在値、閾値、通知レベル、発生時刻
- [ ] 通知メッセージに設備詳細画面へのリンクが含まれる
- [ ] 担当グループのSlackチャンネルに通知される（BR-004: 注意はSlackのみ）※Sprint 1では通知先を初期データとして手動投入
- [ ] 「異常」レベルの場合、全員通知チャンネルにも送信される（BR-006）
- [ ] 通知送信がセンサー値受信から30秒以内に完了する（非機能要件）
