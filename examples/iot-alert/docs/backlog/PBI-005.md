## ストーリー

管理者として、通知先グループ（拠点別×設備種別）を設定したい。なぜなら担当者に適切なアラートだけ届けたいから。

## 受入条件

- [ ] 通知先グループを新規作成できる（グループ名、拠点、設備種別を指定）
- [ ] グループにメンバー（ユーザー）を追加・削除できる
- [ ] メンバーごとにSlack通知ON/OFF、メール通知ON/OFFを設定できる
- [ ] グループに紐づく設備が明示される（拠点×設備種別）
- [ ] アラート発生時、該当グループのメンバーにのみ通知が送信される
- [ ] 設定変更が即時に反映される
