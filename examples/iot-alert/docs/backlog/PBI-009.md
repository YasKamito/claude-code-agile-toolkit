## ストーリー

保全担当者として、設備ごとにセンサーの閾値を設定・変更したい。なぜなら設備ごとに基準が違い、自分で調整する必要があるから。

## 受入条件

- [ ] 設備を選択してセンサー項目ごとの閾値を設定できる（SCR-040）
- [ ] 閾値（警告レベル）を入力すると、注意（80%）と異常（120%）が自動計算・表示される
- [ ] 保存ボタンで閾値が即時反映される（BR-010）
- [ ] 保全担当者が自分で操作できる（IT部門への依頼不要）
- [ ] 不正な値（負数、0、非数値）が入力された場合にバリデーションエラーが表示される
- [ ] 現在の閾値が初期表示される（空の場合は「未設定」と表示）
